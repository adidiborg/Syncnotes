Share Notes

Notes can be shared with generation of a public link that will render a note in Markdown syntax. This is only a view only and not edit feature.



Keyword Extraction

Using NLTK RAKE (Rapid Action Keyword Extraction) algorithm, Keywords can be extracted from notes on basis of Degree and Frequency of word. Further using Unsplash Image API we can use most valued word to get a relevant image for the note.
